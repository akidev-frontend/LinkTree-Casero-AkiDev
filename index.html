Domina Git sin volverte loco

Git puede parecer un laberinto al principio. Branches, commits, push, pull... ¿qué es todo eso?

Subí un vídeo sobre Git sin muchas expectativas… y petó. Más de 3500 vistas y un montón de gente dándome las gracias porque, al parecer, les aclaró muchas dudas.

Vi que la cosa gustó y, aunque mi cuenta va más de frontend, decidí aprovechar el tirón y hacer algo útil: un PDF totalmente gratis con los comandos de Git que realmente necesitas para trabajar sin agobios.

Es directo, práctico y sin rodeos. Lo que me habría gustado tener cuando empecé.

¿Te suena a chino esto?:

"git rebase origin/main && git push --force-with-lease"

Si sabes lo que hace, genial. Si no lo sabes pero quieres entenderlo, este PDF es para ti.
